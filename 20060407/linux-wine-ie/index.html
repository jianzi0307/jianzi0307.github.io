linux下安装运行IE

2006-04-07 /
  • linux
  • wine
 / 点击 /

做出来的页子要在firefox和IE下都测试通过了，才算通过了，可是换了linux以后，就没发用IE来测试页子了，用IE的用户还是占多数，怎么办呢，能不能在linux下安装IE呢？ 答案是肯定的
具体步骤：

  1. 安装wine （http://www.winehq.com/site/download）
    选择符合自己系统的版本，这里选的for ubuntu 的版本（wine_0.9.10-winehq1-2_i386.deb）
    安装方法:

如果你用的其他的发行版，可以下载源码包，自己编译

  1. 安装CrossOver Office （http://www.codeweavers.com/site/products/download_trial/）
    CrossOver Office是用来安装Windows程序的软件

出现图形界面的时候，安装之

  1. 安装 DCOM95.EXE
    安装IE的时候需要它。
    运行CrossOver Office，选择安装DCOM95，下一步，点下载按钮，程序会从微软网站上将其下载下来， 然后安装即可

  2. 安装IE6.0
    步骤同安装DCOM95类似
    安装完毕，桌面上会出现IE久违的e形图标。
    打开的时候会慢些，因为是在wine环境下运行的，而且使用的时候会有些闪烁，不爽，不过仅仅为了测试，足宜……
    折腾了两天，终于可以用linux来工作了 …

全文完。
